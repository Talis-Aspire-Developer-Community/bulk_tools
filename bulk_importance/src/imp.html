This tool was designed for the automated population of a list where bookmarks do not yet exist in TARL.

Configuration

If you wish to assign an item importance, please ensure you have populated the $importanceID variable in user.config.php - please see documentation for more details.

If you have not created your own user.config.php file as a copy of the template in the project root directory do so now!!

Otherwise, this script will give you an error about not being able to find the user.config.php file.
