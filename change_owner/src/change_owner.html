Change Owner

Updates lists with supplied owner (Talis Global User ID or email address)

Configuration

Your file should be either a comma separated file (csv) with:

  • List ID followed by Talis Global User ID
    • or
  • List ID followed by an email address
The file must use either Talis Global User ID or email address for the new owner. Mixing new owner data will cause errors.
If using email addresses, remember to check the Update owner using user email? box below.
