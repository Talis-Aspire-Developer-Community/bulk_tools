Update list time period

Updates lists with the supplied time period.

Configuration

CSV file should include a List ID and its new time period per line.
Report defaults to only writing active time periods to lists, but can be run to udpate lists with inactive time periods.

If unsure what time periods are valid, run this tool in List active time periods mode (upload file not requried).
