Reset List Student Numbers from Hierarchy Nodes

Resets a reading list's student number value to the value held in the list's attached hierarchy node(s)

Default behaviour

By default, if a hierarchy node a list is attached to has no student number information, this tool will keep the existing student numbers linked to nodes on the list, i.e. they will not be overwritten with "0".

If you wish override this behaviour and use the value held in attached hierarchy nodes even if it is "0", tick the box in the configuration section below.
