Edition Updater -

You will need to provide a csv with the following columns: Item IDs to update; old LCN; new LCN; new ISBN-10; new ISBN-13.

You can leave new LCN blank to delete the existing one.

Please include a single ISBN-10 and/or a single ISBN-13

Configuration

If you have not created your own user.config.php file as a copy of the template in the project root directory do so now!!

Otherwise, this script will give you an error about not being able to find the user.config.php file.
